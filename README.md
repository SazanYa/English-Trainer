# Курсовая работа по ООП

В рамках выполнения курсовой работы была поставлена задача – разработать программу – тренажер по английскому.

Функционально программа выполняет:
- Функции преподавателя английского:
  - регистрация и авторизация в приложении,
  - создание тестовых заданий,
  - просмотр результатов выполненных тестовых заданий,
  - выполнение операций над выполненными тестовыми заданиями (удаление, комментирование).
- Функции обучающегося английскому: 
  - регистрация и авторизация в приложении, 
  - выполнение тестовых заданий,
  - просмотр результатов выполненных тестовых заданий,
  - просмотр информации о тестовых заданиях,
  - выполнение поиска тестовых заданий.

При выполнении курсового проекта использовались принципы и приемы ООП. Приложение разработано под **ОС Windows** и представляет собой настольное приложение (desktop). **Язык разработки – С#**. В качестве интерфейса прикладного программирования был выбран обширный API-интерфейс — **Windows Presentation Foundation**. Для работы с базой данных внутри проекта была выбрана технология **Entity Framework 6.0**. Для стилизации приложения была выбрана библиотека **Google Material Design**.
